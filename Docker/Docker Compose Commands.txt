> docker-compose up
To run docker-compose file to start application

> docker-compose up -d
OR
> docker-compose start
To run docker-compose file to start application in background

> docker-compose ps
To list containers started by docker-compose

> docker-compose stop
To stop all containers initiated by docker-compose

> docker-compose stop
To stop all containers initiated by docker-compose

> docker-compose rm
To remove all containers initiated by docker-compose

> docker-compose build
To build image from docker-compose.yml file
